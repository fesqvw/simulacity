# Simulacity

A city simulation.

### Features

- Transport types
    - Foot
    - Car
    - Bus
    - Train
    - Tram
    - Subway
- Building types
    - Residential
    - Industrial
    - Comercial
    - (OPTIONAL) Special industry 
- Land
    - Ground
    - Water 
- (OPTIONAL) Utilities
    - Water
    - Electricity
